# Madison Reed Image Gallery

## Setup
Install mise for automatic language agnostic global dependencies installation
[Learn more about mise](https://mise.jdx.dev/)
```
curl https://mise.run | sh
```

Automatic first setup

```bash
mise dev
```

```bash
# pnpm
pnpm install

#dev
pnpm dev
```
